## 需求分析

### 原有需求

1. **会议块上沿对准时间轴**：会议块的上沿应该对齐 hour-row 的 border-bottom
2. **时间轴高度调整**：增加时间轴高度
3. **会议块文字样式**：参考设计图优化标题和时间显示
4. **日期选择器优化**：宽度增加、显示完整月日、预约数改为圆圈徽章

### 新增需求

1. **时间轴高度**：HOUR_HEIGHT 从 100rpx 增加到 **200rpx**
2. **半点显示**：时间轴除显示整点外，还显示半点，以**虚线**显示
3. **会议块拖拽**：

- 支持拖拽手势在时间轴上直接修改会议时间
- 拖拽时**吸附到整点或半点**（每30分钟一个刻度）

## 技术方案

### 修改文件

| 文件 | 修改内容 |
| --- | --- |
| `meeting-room.js` | HOUR_HEIGHT=200、半点数据、拖拽逻辑、日期格式 |
| `meeting-room.wxml` | 半点虚线、会议块拖拽区域、日期显示、预约数徽章 |
| `meeting-room.wxss` | 时间轴高度、半点虚线样式、拖拽反馈样式、日期块样式 |


### 关键实现

#### 1. 时间轴改造（200rpx + 半点虚线）

**WXML 结构**：

```
<view class="timeline-grid">
  <view class="hour-row" wx:for="{{hours}}" wx:key="*this" id="hour-{{item}}">
    <view class="hour-label">{{item < 10 ? '0' + item : item}}:00</view>
    <!-- 半点虚线 -->
    <view class="half-hour-line"></view>
  </view>
</view>
```

**WXSS 样式**：

- `.hour-row`: height: 200rpx
- `.half-hour-line`: position: absolute, top: 100rpx, left: 88rpx, right: 32rpx, border-top: 1rpx dashed #E2E8F0

#### 2. 会议块拖拽功能

**拖拽区域划分**：

- **上半部分（顶部 20%）**：拖拽调整开始时间
- **中间部分**：整体移动（开始和结束时间同时调整）
- **下半部分（底部 20%）**：拖拽调整结束时间

**吸附逻辑**：

```js
// 将分钟吸附到 0 或 30
function snapToHalfHour(minutes) {
  return minutes < 15 ? 0 : (minutes < 45 ? 30 : 60)
}
```

**拖拽状态**：

- draggingReservation: 正在拖拽的预约
- dragType: 'start' | 'end' | 'move'
- previewTime: 预览时间

#### 3. 日期选择器优化

- .date-item width: 100rpx → 120rpx
- 日期显示格式：`${month}月${day}日`
- .date-count 改为圆圈徽章：width/height 36rpx，圆形背景，居中显示数字
