## 产品概述

护照领用功能，支持符合条件的用户申请借用护照，经过工作流审批后由管理员确认借出。

## 核心功能

1. **权限控制**：仅'馆领导'、'部门负责人'、'馆员'、'工勤'、'物业'角色可使用，其他人提示无权限
2. **护照借出申请**：填写领用人姓名（多个空格分隔）、借用日期（默认今日）、预计归还日期（可选）、借用事由
3. **工作流审批**：三步审批流程

- 第一步：申请人同部门的部门负责人审批
- 第二步：馆领导审批
- 第三步：管理员确认借出

4. **领用状态展示**：显示当前是否已领用护照，已领用时显示领用信息
5. **领用历史**：分页展示历史申请记录，点击查看详情弹窗
6. **首页入口**：在home页常用功能模块添加图标入口

## 技术栈

- 小程序框架：微信小程序原生框架
- 云开发：CloudBase 云函数 + NoSQL 数据库
- 工作流引擎：现有 workflowEngine 云函数

## 实现方案

### 1. 工作流引擎扩展

在现有 `workflowEngine` 中新增 `approverType: 'dept_head'` 类型，实现同部门负责人审批人解析逻辑：

```javascript
// resolveApprovers 函数新增分支
case 'dept_head':
  // 获取申请人部门
  const applicantRes = await usersCollection.where({
    openid: businessData.applicantId
  }).limit(1).get()
  
  const applicantDept = applicantRes.data[0]?.department
  
  // 查找同部门的部门负责人
  const deptHeadsRes = await usersCollection.where({
    role: '部门负责人',
    department: applicantDept,
    status: 'approved'
  }).get()
  
  return deptHeadsRes.data.map(user => ({
    id: user.openid,
    name: user.name
  }))
```

### 2. 工作流模板配置

在 `initWorkflowDB` 云函数中添加护照领用模板：

```javascript
{
  name: '护照领用审批',
  code: 'passport_application',
  version: 1,
  steps: [
    {
      stepNo: 1,
      stepName: '部门负责人审批',
      stepType: 'serial',
      approverType: 'dept_head',
      canReject: true,
      canReturn: true
    },
    {
      stepNo: 2,
      stepName: '馆领导审批',
      stepType: 'serial',
      approverType: 'role',
      approverConfig: { roleIds: ['library_leader'] }
    },
    {
      stepNo: 3,
      stepName: '管理员借出',
      stepType: 'serial',
      approverType: 'role',
      approverConfig: { roleIds: ['admin'] }
    }
  ]
}
```

### 3. 权限配置

通过 `permissionManager` 配置功能权限：

- featureKey: `passport_application`
- enabledRoles: ['馆领导', '部门负责人', '馆员', '工勤', '物业']

### 4. 领用状态判断

```javascript
// 已领用条件
{
  orderType: 'passport_application',
  workflowStatus: 'completed',
  'businessData.borrowerOpenids': openid,
  'businessData.returnedAt': null // 未归还
}
```

## 设计风格

延续现有小程序办公系统风格，采用渐变色头部 + 卡片式布局。

## 页面布局

参考 trip-report 页面的状态卡片 + 历史列表布局，help 页面的标题区域设计。

### 页面结构

1. **渐变色头部**：标题"护照领用" + 右侧"护照借出"按钮
2. **领用状态卡片**：玻璃态卡片，显示当前领用状态
3. **领用历史列表**：按月分组，分页加载，点击弹出详情

### 弹出表单

底部弹出表单，包含：

- 领用护照输入框（必填，多个姓名空格分隔）
- 借用日期选择器（默认今日）
- 预计归还日期选择器（可选）
- 借用事由文本框（必填）
- 提交申请按钮
