## 产品概述

为小程序home页添加"预约会议室"新功能，图标添加在常用功能模块。用户可通过该功能预约会议室、查看会议室预约情况。

## 核心功能

### 1. 会议室选择

- 横向可滑动列表显示所有会议室（260会议室、3楼会议室等）
- 点击切换当前选中的会议室

### 2. 日期选择

- 从今天起往后7天的日期横向列表（可左右滑动）
- 每个日期块显示：星期、日期、当日已预约会议数
- 点击切换选中日期

### 3. 时间轴会议列表

- 类似calendar页的时间轴日程列表设计
- 0-24小时网格布局
- 紫色线代表当前时间
- 白色圆角矩形显示已预约时间段
- 会议块内显示：会议标题、会议时间段、备注
- 点击会议弹出详情页
- 长按时间轴空白处添加新会议预约

### 4. 会议预约表单

- 会议标题（输入）
- 会议室（选择，默认当前选中会议室）
- 开始时间（选择）
- 结束时间（选择）
- 备注（输入，可选）

### 5. 会议详情页

- 显示会议所有字段信息
- 创建者可编辑/删除

## 技术栈

- 微信小程序原生开发（WXML/WXSS/JS）
- CloudBase 云开发（云函数 + NoSQL数据库）
- 复用现有项目架构和组件

## 实现方案

### 架构设计

采用独立页面方案，在home页添加入口，点击后跳转到独立的会议室预约页面：

```
home页（添加入口图标）
    ↓
meeting-room页面（会议室预约主页面）
    ├── 会议室选择器（横向滑动）
    ├── 日期选择器（7天横向滑动）
    ├── 时间轴会议列表（核心功能）
    │   └── 长按添加功能
    ├── 会议详情弹窗
    └── 会议编辑/创建弹窗
```

### 数据库设计

**集合名**: `meeting_room_reservations`

**字段定义**:

```javascript
{
  _id: string,              // 系统自动生成
  _openid: string,          // 创建者openid（安全规则用）
  title: string,            // 会议标题（必填）
  roomId: string,           // 会议室ID（必填）
  roomName: string,         // 会议室名称（冗余存储，显示用）
  date: string,             // 预约日期 YYYY-MM-DD（必填）
  startTime: string,        // 开始时间 HH:mm（必填）
  endTime: string,          // 结束时间 HH:mm（必填）
  description: string,      // 备注（可选）
  creatorName: string,      // 创建者姓名
  createdAt: number,        // 创建时间戳
  updatedAt: number         // 更新时间戳
}
```

**安全规则**: `READONLY`（所有用户可读，仅创建者可写）

**索引**:

- `roomId` + `date`（复合索引，查询优化）
- `date`（单字段索引）

### 云函数设计

**云函数名**: `meetingRoomManager`

**Actions**:

| Action | 功能 | 参数 |
| --- | --- | --- |
| `getMeetingRoomReservations` | 获取指定会议室、日期的预约列表 | roomId, date |
| `createMeetingRoomReservation` | 创建新预约 | 所有字段 |
| `updateMeetingRoomReservation` | 更新预约 | reservationId, 更新字段 |
| `deleteMeetingRoomReservation` | 删除预约 | reservationId |
| `checkMeetingRoomConflict` | 检查时间冲突 | roomId, date, startTime, endTime, excludeId |


> **命名说明**: Action 名称包含 `MeetingRoom` 前缀，为后续添加其他预约功能（如车辆预约、设备预约）预留扩展空间。

### 前端实现要点

1. **时间轴布局算法**

- 简化版布局算法（单列显示）
- 每小时高度100rpx
- **无需多会议重叠并排显示**：同一会议室同一时间段只能有一场会议
- 会议块按开始时间排序，垂直排列

2. **时间冲突检测**

- 前端预检：提交前检查时间段是否已被预约
- 后端校验：云函数二次确认，防止并发冲突

3. **长按添加功能**

- 使用`bindlongpress`事件
- 计算长按位置对应的时间点
- 自动填充默认开始/结束时间

4. **当前时间指示线**

- 每分钟更新一次位置
- 自动滚动到当前时间附近

## 页面设计架构

### 整体布局（从上到下）

**1. 页面头部**

- 左侧：标题"预约会议室"（深色字体，24px，600字重）
- 右侧：圆形加号按钮（紫色渐变背景，白色+号图标，直径80rpx）
- 背景：浅紫色渐变（#E8E4FF 到 #F5F3FF）

**2. 会议室选择器**

- 横向可滑动scroll-view
- 会议室标签：圆角矩形（24rpx圆角），选中态为深紫色背景+白色文字，未选中为浅灰背景+深灰文字
- 间距：标签间16rpx，整体左右32rpx内边距
- 默认选中第一个会议室

**3. 日期选择器**

- 横向可滑动scroll-view（与会议室选择器独立）
- 日期块：垂直布局，从上到下依次为：
- 星期（缩写，如"周日"，14px，灰色）
- 日期（数字，如"24"，20px，深色，600字重）
- 预约数（小圆点+数字，如"4"，12px，灰色）
- 选中态：浅紫色背景圆角矩形（24rpx圆角），日期数字变为紫色
- 默认选中今天

**4. 时间轴会议列表**

- 垂直滚动scroll-view
- 左侧：小时标签（00:00-24:00，每1小时一行，14px，灰色）
- 右侧：会议块
- 白色圆角矩形背景（20rpx圆角，轻微阴影）
- 会议标题（16px，深色，600字重）
- 时间段（图标+时间范围，14px，灰色）
- 备注（如有，14px，灰色，单行省略）
- 当前时间线：紫色水平线（2px粗），左侧带时间标签
- 长按空白区域触发添加

### 弹窗设计

**会议详情弹窗**

- 底部弹出卡片（圆角32rpx）
- 头部：关闭按钮、标题"会议详情"、编辑按钮
- 内容区：会议标题、时间信息、会议室、备注、创建者
- 创建者显示删除按钮

**会议编辑弹窗**

- 底部弹出卡片（圆角32rpx）
- 头部：取消按钮、标题（新建/编辑会议）、完成按钮
- 表单区：标题输入、会议室选择、开始时间、结束时间、备注输入
