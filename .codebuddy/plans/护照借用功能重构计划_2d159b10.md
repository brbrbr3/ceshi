## 产品概述

护照借用管理功能重构，引入Tab页架构，优化工作流程，新增护照信息维护和到期提醒功能。

## 核心功能

1. **Tab页架构**：护照借出、待借护照（管理员）、在借护照（管理员）、护照信息维护四个Tab页
2. **工作流简化**：删除管理员借出步骤，馆领导审批后状态为"已批准，待借出"
3. **管理员操作**：待借护照列表可执行"借出"操作；在借护照列表可执行"收回"或"不再收回"操作
4. **护照信息维护**：用户填写护照信息（姓名、性别、护照号、签发日期、有效期至），支持添加多本护照
5. **过期提醒**：护照有效期不足半年时推送通知提醒用户

## 技术栈

- 微信小程序原生开发 + CloudBase 云开发
- 云函数：wx-server-sdk
- 数据库：NoSQL 文档数据库

## 实现方案

### 1. 数据库设计

**新增集合 `passport_info`**：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| openid | string | 创建者openid |
| ownerName | string | 护照持有人姓名 |
| gender | string | 性别（男/女） |
| passportNo | string | 护照号 |
| issueDate | string | 签发日期（YYYY-MM-DD） |
| expiryDate | string | 有效期至（YYYY-MM-DD） |
| createdAt | number | 创建时间戳 |
| updatedAt | number | 更新时间戳 |


**修改 `passport_records` 状态逻辑**：

- `approved`：已批准，待借出（新增）
- `borrowed`：已借出
- `returned`：已归还
- `not_returned`：不再收回（新增）

新增字段：

- `borrowedAt`：实际借出时间
- `borrowedBy`：借出操作人openid
- `borrowedByName`：借出操作人姓名

### 2. 工作流修改

**修改工作流模板**：删除第3步"管理员借出"，保留2步流程

**修改工作流引擎**：馆领导审批通过后创建 `passport_records` 记录，状态为 `approved`

### 3. 云函数修改

**passportManager 云函数新增 action**：

- `getPassportInfoList`：获取用户护照信息列表
- `addPassportInfo`：添加护照信息
- `updatePassportInfo`：更新护照信息
- `deletePassportInfo`：删除护照信息
- `borrowPassport`：管理员借出操作
- `returnPassport`：管理员收回操作
- `markNotReturned`：标记不再收回
- `getApprovedList`：获取待借护照列表

**新增定时触发器云函数**：`passportExpiryChecker`，每日执行，检查护照有效期并发送通知

### 4. 前端重构

页面采用Tab架构，实现四个功能模块，遵循项目编码规范中的Tab切换加载规则。

## 注意事项

1. 工作流模板修改后需重新运行 `initWorkflowDB` 云函数初始化
2. 已存在的老数据需兼容处理（status字段可能为空，默认视为borrowed）
3. Tab切换遵循规范：先清空列表 → 显示loading → 重置分页 → 加载数据
4. 权限检查使用 `app.checkUserRegistration()` 获取用户信息判断 `isAdmin`

## 设计风格

采用现代化小程序设计风格，卡片式布局，Tab页切换流畅。遵循项目现有的UI规范，渐变色头部配合白色卡片。

## 页面布局

### 头部区域

渐变色背景标题栏，显示"护照借用"标题，右侧保留"申请借出护照"按钮（仅在护照借出Tab显示）

### Tab栏

固定在头部下方，采用下划线激活指示器样式：

- 护照借出（默认激活）
- 待借护照（仅管理员可见）
- 在借护照（仅管理员可见）
- 护照信息维护

### 内容区域

卡片式列表布局，各Tab内容：

1. **护照借出Tab**：保留当前页面内容（借用状态卡片 + 申请记录列表）
2. **待借护照Tab**：已批准待借出申请列表，点击显示详情弹窗+借出按钮
3. **在借护照Tab**：在借护照列表，点击显示详情弹窗+收回/不再收回按钮
4. **护照信息维护Tab**：护照列表+添加按钮，点击弹出表单

### 弹窗设计

底部弹出式弹窗，显示详情信息和操作按钮
